Free Hosting! Find out how

Author: Miguel Teixeira
2015-04-08
Tags: Resources, Hosting


TweetFollow @idea_ninja

Are you spending money on hosting ? No more! You can use github to host your startup website for free right now. Read this article to learn how.

So you created a website. Now is time to roll it out. This is what you need to do before following these instructions:

  • Have a domain
  • Have a website built
  • Register on Github.com

Step 1

Create a repository on Github with your website. Note that your branch now is called Master.

Step 2

Create a new branched named gh-pages.

Step 3

Deploy your website. Note you should deploy your DOM files and folders.

Step 4

Test your website, through the following URL [username].github.io/[repository name]

Step 5

Add a File named CNAME (all Caps and no extension) and in that file write down your domain.

Step 6

Go to your domain and edit the Zone file or change it to this:

or this:
@ 10800 IN A 192.30.252.153
@ 10800 IN A 192.30.252.154
www 10800 IN CNAME @

Step 7

Make sure that your domain is using the zone file you edited/created.

Note: it may take a few hours before the website is available. These new settings need to be propagated through the ISP, so don't worry if you cannot see your site right after you completed these 7 steps.
