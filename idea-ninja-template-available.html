Idea Ninja template available

Author: Miguel Teixeira
2015-04-08
Tags: Website, template, css, Editorial


Tweet Follow @idea_ninja

Idea Ninja has made its site template available to you, so you can use it and abuse it when building your startup's website. You can download it, trunk it, review it and add more features.

From the get-go Ideaninja shares its template to you. This template is a living organism, it will evolve as the site evolves and all will continue to be shared.

You can find it in this github repository You can download it freely, trunk it, make it evolve.

This template was initial developed using Gulp with Compass, Susy and Breakpoint among others.
